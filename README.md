## 🚀 Quick Start

```bash
# Clone and setup
git clone https://github.com/ubaidsk/ubaidsk.github.io.git
cd ubaidsk.github.io
git submodule update --init --recursive

# Run locally
hugo server -D

# Build for production
hugo --minify
```

Visit [http://localhost:1313/](http://localhost:1313/) during development.
