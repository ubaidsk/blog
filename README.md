## 🚀 Quick Start

```bash
# Clone and setup
git clone https://github.com/ubaidsk/blog.git
cd blog
git submodule update --init --recursive

# Run locally
hugo server -D

# Build for production
hugo --minify
```

Visit [http://localhost:1313/blog/](http://localhost:1313/blog/) during development.
